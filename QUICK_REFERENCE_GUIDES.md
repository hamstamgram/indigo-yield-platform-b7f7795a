## 1. How to Make Your First Deposit

**Time Required**: 15-30 minutes
**Difficulty**: Easy

### Before You Start
✓ Account fully verified and Active
✓ KYC documents approved
✓ Bank or wallet information ready
✓ Know minimum investment amount

### Step-by-Step

**Step 1: Check Requirements**
- [ ] Log into your account
- [ ] Go to Dashboard
- [ ] Verify account status shows "Active"
- [ ] Note minimum deposit for your chosen fund

**Step 2: Initiate Deposit**
- [ ] Contact your fund administrator for deposit instructions
- [ ] Receive wire/ACH/crypto instructions via secure email
- [ ] Note reference number or memo to include
- [ ] Write down exact amount you'll deposit

**Step 3: Send Funds**
- [ ] Initiate transfer from your bank/exchange
- [ ] Include reference number in memo/notes field
- [ ] Save confirmation number/transaction hash
- [ ] Take screenshot of confirmation

**Step 4: Notify Platform** (if required)
- [ ] Email confirmation to support team
- [ ] Include: transaction reference, amount, date
- [ ] Attach confirmation screenshot

**Step 5: Track Deposit**
- [ ] Check Transactions page for pending deposit
- [ ] Wait for confirmation (3-5 business days typically)
- [ ] Verify balance updates in Dashboard
- [ ] Confirm you received email notification

### Expected Timeline
- **ACH**: 3-5 business days
- **Wire**: 1-2 business days
- **Crypto**: Hours to 1 day

### What If Something Goes Wrong?
- Deposit not showing after 5 business days → Contact support
- Wrong amount sent → Contact support immediately
- Missing reference number → Email support with details

## 2. How to Request a Withdrawal

**Time Required**: 10 minutes
**Difficulty**: Easy

### Before You Start
✓ Sufficient available balance
✓ Lock-up period met (if applicable)
✓ Bank/wallet information on file
✓ No compliance holds on account

### Step-by-Step

**Step 1: Navigate to Withdrawals**
- [ ] Log into your account
- [ ] Click "Withdrawals" in main menu
- [ ] Review your current positions

**Step 2: Click "New Withdrawal"**
- [ ] Locate "New Withdrawal" button (top-right)
- [ ] Click to open withdrawal form

**Step 3: Select Fund**
- [ ] Open "Select Fund" dropdown
- [ ] Review available balance for each fund
- [ ] Choose fund to withdraw from
- [ ] Note maximum withdrawal amount shown

**Step 4: Enter Amount**
- [ ] Enter withdrawal amount in USD
- [ ] Verify amount is ≤ available balance
- [ ] Use decimal point for cents if needed
- [ ] Double-check for typos

**Step 5: Choose Type**
- [ ] Select "Partial" to keep position open
  - OR -
- [ ] Select "Full" to close position completely

**Step 6: Add Notes** (Optional but Recommended)
- [ ] Enter any special instructions
- [ ] Note preferred payment method
- [ ] Mention urgency if time-sensitive
- [ ] Add contact info if traveling

**Step 7: Review and Submit**
- [ ] Verify all information correct
- [ ] Double-check withdrawal amount
- [ ] Click "Submit Request"
- [ ] Wait for confirmation message
- [ ] Note request ID for tracking

**Step 8: Track Status**
- [ ] Request appears in withdrawal history
- [ ] Status: "Pending" (awaiting admin review)
- [ ] Check email for status updates
- [ ] Monitor withdrawals page for progress

### What Happens Next
1. **Admin Review** (1-2 business days)
2. **Approval** or request for more info
3. **Processing** (1-3 days)
4. **Completion** and funds sent
5. **Confirmation** email received

### Expected Timeline
- **Total**: 5-7 business days typically
- **Urgent**: Contact support for expedited processing

## 3. How to Read Your Monthly Statement

**Time Required**: 15 minutes
**Difficulty**: Medium

### Accessing Your Statement

**Step 1: Navigate to Documents**
- [ ] Log into account
- [ ] Click "Documents" in main menu
- [ ] Look for "Monthly Statements" section

**Step 2: Find Correct Statement**
- [ ] Statements organized by year/month
- [ ] Click desired statement
- [ ] PDF opens in browser or downloads

### Understanding Each Section

**Page 1: Account Summary**
- [ ] **Beginning Balance**: Value at start of month
- [ ] **Ending Balance**: Value at end of month
- [ ] **Net Change**: Difference (+ gain / - loss)
- [ ] **Performance %**: Monthly return percentage

**Page 2: Transaction Detail**
- [ ] Review each transaction
- [ ] Check dates are correct
- [ ] Verify amounts match your records
- [ ] Note any fees charged

**Page 3: Holdings Detail**
- [ ] **Position**: Each fund you own
- [ ] **Shares**: Quantity owned
- [ ] **Price**: NAV per share
- [ ] **Value**: Total position value
- [ ] **Allocation**: Percentage of portfolio

**Page 4: Performance**
- [ ] **Month**: This month's return
- [ ] **YTD**: Year-to-date return
- [ ] **Inception**: Total return since you started
- [ ] **Benchmark**: Comparison to market

**Page 5: Fees**
- [ ] Management fees charged
- [ ] Performance fees (if earned)
- [ ] Other fees
- [ ] Year-to-date total fees

### What to Check

**Verify Accuracy**:
- [ ] All transactions you made are listed
- [ ] No unauthorized transactions
- [ ] Amounts are correct
- [ ] Fees match expectations
- [ ] Ending balance matches dashboard

**Look For**:
- [ ] Performance vs. goals
- [ ] Fee impact on returns
- [ ] Allocation drift
- [ ] Unexpected changes

### If You Find Errors
1. [ ] Note specific error (page, line item)
2. [ ] Gather supporting documents
3. [ ] Contact support within 30 days
4. [ ] Request corrected statement if needed

### Save for Records
- [ ] Download PDF to secure location
- [ ] Back up to cloud storage
- [ ] Keep for tax preparation
- [ ] Retain for at least 7 years

**Difficulty**: Easy

- Quarterly estimated taxes
- Tax planning sessions
- IRS audit

### Before You Start
✓ Tax year ended (generate after December 31)
✓ All year's transactions complete
✓ Official tax forms received (1099, K-1)

- [ ] Click "Analytics" or "Reports" in menu
- [ ] Look for "Tax Reports" section

- [ ] Verify correct tax year selected
- [ ] Review report options

**Step 3: Generate Report**
- [ ] Click "Generate Tax Report"
- [ ] Wait for processing (30-60 seconds)
- [ ] Report displays on screen

**Step 4: Review Report Contents**
- [ ] **Realized Gains/Losses**: From sales/withdrawals
- [ ] **Dividend/Interest Income**: Yield earned
- [ ] **Fees Paid**: Potentially deductible
- [ ] **Cost Basis**: For each position

- [ ] Go to Documents → Tax Forms
- [ ] Download Form 1099 (or K-1)
- [ ] Received by January 31st typically

### Important Notes

**For Your Tax Preparer**:
- [ ] Provide official tax forms (1099/K-1)
- [ ] Include tax report as supplemental info
- [ ] Share transaction history if requested
- [ ] Note any special situations

**Remember**:
- Platform report is for reference only
- Official tax forms used for actual filing
- Consult tax professional for advice
- Keep all documents for 7+ years

**Difficulty**: Easy

### When to Update
- Address change
- Phone number change
- Email change
- Name change (legal)
- Bank information update

### Step-by-Step

**Step 1: Navigate to Profile**
- [ ] Log into account
- [ ] Click your name (top-right) or "Account"
- [ ] Select "Profile Settings"

**Step 2: Edit Information**
- [ ] Click "Edit" button
- [ ] Update desired fields:
  - [ ] First name
  - [ ] Last name
  - [ ] Phone number
  - [ ] Mailing address
  - [ ] Email (requires verification)
- [ ] Leave unchanged fields as-is

**Step 3: Save Changes**
- [ ] Review all information
- [ ] Click "Save Changes"
- [ ] Wait for confirmation message

**Step 4: Verify Updates** (if applicable)
- [ ] **Email Change**: Check new email for verification link
- [ ] Click verification link
- [ ] Confirm email updated
- [ ] **Phone Change**: Enter verification code sent via SMS

### Special Cases

**Legal Name Change**:
- [ ] Cannot self-update in most cases
- [ ] Contact support
- [ ] Provide legal documentation (marriage certificate, court order)

- [ ] Navigate to Settings → Payment Methods
- [ ] Click "Add Bank Account"
- [ ] Enter routing and account numbers

- [ ] Contact support (cannot self-update)
- [ ] Provide documentation
- [ ] Security verification required

### Verification Timeline
- **Phone**: Instant (SMS code)
- **Email**: 5-15 minutes
- **Bank**: 1-3 business days (micro-deposits)
- **Legal Name**: 3-5 business days (manual review)

## 6. How to Enable 2FA (Two-Factor Authentication)

**Time Required**: 10 minutes
**Difficulty**: Easy

### Why Enable 2FA?
- ✓ Protects against unauthorized access
- ✓ Required for large withdrawals
- ✓ Industry best practice
- ✓ Adds minimal inconvenience

### What You'll Need
- Smartphone
- Authenticator app (Google Authenticator, Authy, etc.)
  - OR -
- Phone number for SMS codes

### Step-by-Step (Authenticator App - Recommended)

**Step 1: Navigate to Security Settings**
- [ ] Log into account
- [ ] Go to Account → Security Settings
- [ ] Find "Two-Factor Authentication" section

**Step 2: Choose Method**
- [ ] Click "Enable 2FA"
- [ ] Select "Authenticator App" (most secure)
- [ ] Click "Continue"

**Step 3: Install Authenticator App** (if needed)
- [ ] Download from app store:
  - [ ] Google Authenticator (iOS/Android)
  - [ ] Authy (iOS/Android)
  - [ ] Microsoft Authenticator (iOS/Android)
- [ ] Install and open app

**Step 4: Scan QR Code**
- [ ] QR code displays on screen
- [ ] Open authenticator app
- [ ] Tap "+" or "Add Account"
- [ ] Scan QR code with phone camera
- [ ] Account added to app

**Step 5: Enter Verification Code**
- [ ] Authenticator app shows 6-digit code
- [ ] Code changes every 30 seconds
- [ ] Enter current code on platform
- [ ] Click "Verify"

**Step 6: Save Backup Codes**
- [ ] Platform shows 10 backup codes
- [ ] **CRITICAL**: Write down or download
- [ ] Store in secure location (password manager or safe)
- [ ] Each code works only once
- [ ] Click "I've saved my backup codes"

**Step 7: Confirm 2FA Enabled**
- [ ] Success message appears
- [ ] 2FA badge shown in security settings
- [ ] Confirmation email received

### Using 2FA at Login

**Every Login**:
1. [ ] Enter email and password
2. [ ] Platform asks for 2FA code
3. [ ] Open authenticator app
4. [ ] View 6-digit code for Indigo platform
5. [ ] Enter code
6. [ ] Click "Verify"
7. [ ] Successfully logged in

### If You Lose Phone

**With Backup Codes**:
1. [ ] Use one backup code to log in
2. [ ] Go to Security Settings
3. [ ] Click "Reset 2FA"
4. [ ] Set up with new phone
5. [ ] Generate new backup codes

**Without Backup Codes**:
1. [ ] Contact support immediately
2. [ ] Verify identity (additional steps required)
3. [ ] Support resets 2FA
4. [ ] Set up again with new phone

### Alternative: SMS (Less Secure)

**If Using SMS**:
- [ ] Select "SMS" instead of app
- [ ] Enter phone number
- [ ] Receive verification code via text
- [ ] Enter code to enable
- [ ] Codes sent via text at each login

**Note**: Authenticator app preferred over SMS (more secure, works without cell service).

## 7. How to Contact Support

**Time Required**: 5 minutes
**Difficulty**: Easy

### Support Options

**Help Center** (Self-Service)
- Available 24/7
- Search for answers
- Step-by-step guides
- Video tutorials

**Support Ticket** (Email-Based)
- Response: 24-48 hours
- For non-urgent issues
- Detailed questions
- Document requests

**Live Chat** (If Available)
- Business hours only
- Quick questions
- Real-time guidance
- Immediate help

**Phone** (If Available)
- Urgent issues
- Complex problems
- Prefer verbal communication

### How to Submit a Support Ticket

**Step 1: Navigate to Support**
- [ ] Log into account
- [ ] Click "Support" in main menu
- [ ] Click "Submit Ticket"

**Step 2: Select Category**
- [ ] Choose most relevant category:
  - [ ] Account & Verification
  - [ ] Deposits & Transactions
  - [ ] Withdrawals
  - [ ] Technical Issues
  - [ ] Documents & Statements
  - [ ] Other

**Step 3: Fill Out Form**
- [ ] **Subject**: Brief description (50 characters)
  - Example: "Deposit not showing in account"
- [ ] **Description**: Detailed explanation
  - What happened?
  - What did you expect?
  - Steps you took
  - Error messages
  - When did it occur?

**Step 4: Attach Files** (if helpful)
- [ ] Screenshots showing issue
- [ ] Transaction confirmations
- [ ] Error messages
- [ ] Documents (max 10MB per file)

**Step 5: Submit**
- [ ] Review for accuracy
- [ ] Click "Submit Ticket"
- [ ] Note ticket number
- [ ] Check email for confirmation

**Step 6: Track Progress**
- [ ] Go to Support → My Tickets
- [ ] View ticket status:
  - **Open**: Being reviewed
  - **In Progress**: Agent working on it
  - **Waiting for Customer**: Need info from you
  - **Resolved**: Issue fixed
- [ ] Respond to agent messages
- [ ] Close ticket when satisfied

### Tips for Faster Resolution

**Be Specific**:
- ✓ Include account email
- ✓ Provide transaction IDs
- ✓ Note exact dates and amounts
- ✓ Attach relevant screenshots
- ✗ Vague descriptions

**Be Professional**:
- ✓ Polite and courteous
- ✓ Clear and concise
- ✓ Stick to facts
- ✗ Angry or demanding

**Be Patient**:
- ✓ Allow 24-48 hours for response
- ✓ Check spam folder for replies
- ✓ Respond promptly to requests for info
- ✗ Submit duplicate tickets

### Urgent Issues

**If Urgent**:
1. [ ] Mark ticket as "High Priority"
2. [ ] Call phone support (if available)
3. [ ] Explain urgency clearly
4. [ ] Provide all details upfront

**What's Considered Urgent**:
- Account security breach
- Unauthorized transactions
- Login issues preventing access
- Time-sensitive withdrawal
- System error preventing critical action

**Not Urgent**:
- General questions
- Historical statements
- Profile updates
- Feature requests

## 9. ADMIN: How to Approve a New Investor

**Time Required**: 20-30 minutes
**Difficulty**: Medium

### Before You Start
✓ Admin access to platform
✓ Understanding of KYC/AML requirements
✓ Access to sanctions screening tools
✓ Authority to approve accounts

### Step-by-Step

**Step 1: Navigate to Pending Investors**
- [ ] Log in with admin credentials
- [ ] Go to Admin → Investors
- [ ] Filter by Status: "Pending"
- [ ] See list of investors awaiting approval

**Step 2: Select Investor to Review**
- [ ] Click investor name
- [ ] Opens investor detail page

**Step 3: Review Application**
- [ ] **Personal Info Tab**:
  - [ ] Verify name, email, phone complete
  - [ ] Check address provided
  - [ ] Note investor type (Individual, Entity, etc.)
- [ ] **Investment Profile Tab**:
  - [ ] Review accreditation status
  - [ ] Check investment objectives
  - [ ] Note risk tolerance

**Step 4: Verify KYC Documents**
- [ ] Click "Documents" tab
- [ ] Review uploaded documents:
  - [ ] Government ID (passport, driver's license)
    - [ ] Photo clear and legible
    - [ ] Name matches application
    - [ ] Not expired
  - [ ] Proof of address
    - [ ] Utility bill or bank statement
    - [ ] Less than 3 months old
    - [ ] Address matches application
  - [ ] Accreditation proof (if applicable)
    - [ ] Tax returns for income method
    - [ ] Financial statements for net worth method
    - [ ] Professional license for designation method

**Step 5: Sanctions Screening**
- [ ] Click "Run Sanctions Check"
- [ ] System checks against OFAC lists
- [ ] Review results:
  - [ ] ✅ No match → Proceed
  - [ ] ⚠️ Possible match → Manual review
  - [ ] 🚫 Confirmed match → STOP, escalate

**Step 6: Risk Assessment**
- [ ] Review investor profile
- [ ] Check for red flags:
  - [ ] Unusual geographic source
  - [ ] Inconsistent information
  - [ ] Politically exposed person (PEP)
  - [ ] High-risk jurisdiction
- [ ] Document assessment in notes

**Step 7: Decision**

**If All Checks Pass**:
- [ ] Click "Approve Account" button
- [ ] Add approval notes documenting:
  - [ ] KYC documents reviewed and verified
  - [ ] Sanctions screening passed
  - [ ] No red flags identified
  - [ ] Date and time of approval
  - [ ] Your name/ID
- [ ] Click "Confirm Approval"
- [ ] Status changes to "Active"
- [ ] Welcome email sent automatically

**If More Info Needed**:
- [ ] Click "Request Additional Information"
- [ ] Check missing items:
  - [ ] Better quality ID photo
  - [ ] More recent proof of address
  - [ ] Additional accreditation docs
  - [ ] Clarification on source of funds
- [ ] Add custom message explaining what's needed
- [ ] Click "Send Request"
- [ ] Email sent to investor
- [ ] Status remains "Pending"
- [ ] Set reminder to follow up in 7 days

**If Denied**:
- [ ] Click "Deny Application"
- [ ] Select reason:
  - [ ] Failed sanctions screening
  - [ ] Insufficient documentation
  - [ ] Does not meet requirements
  - [ ] Other (specify)
- [ ] Add detailed explanation in notes
- [ ] Click "Confirm Denial"
- [ ] Status changes to "Denied"
- [ ] Account cannot be reopened (must reapply)

**Step 8: Document Decision**
- [ ] Go to "Notes" tab
- [ ] Add admin note with:
  - [ ] Approval/denial decision
  - [ ] Key factors considered
  - [ ] Any special conditions
  - [ ] Follow-up actions needed
- [ ] Save note

**Step 9: Follow-Up Actions**
- [ ] If approved:
  - [ ] Assign account manager (if applicable)
  - [ ] Add to mailing lists
  - [ ] Schedule welcome call (for VIPs)
- [ ] If more info requested:
  - [ ] Set calendar reminder for follow-up
  - [ ] Monitor for investor response
- [ ] If denied:
  - [ ] File documentation
  - [ ] No further action

### Quality Control

**Approval Checklist**:
- [ ] ✓ All personal information complete
- [ ] ✓ Valid government ID provided
- [ ] ✓ Proof of address verified
- [ ] ✓ Accreditation verified (if required)
- [ ] ✓ Sanctions screening passed
- [ ] ✓ No red flags identified
- [ ] ✓ Meets minimum requirements
- [ ] ✓ Approval documented in notes

### Escalation

**Escalate If**:
- Sanctions screening match
- Politically exposed person (PEP)
- Inconsistent or suspicious information
- High-value investor (>$1M)
- Uncertain about any aspect

**Escalation Process**:
1. [ ] Document concern in notes
2. [ ] Contact compliance officer
3. [ ] Await guidance before approving
4. [ ] Proceed per instructions

## 10. ADMIN: How to Process a Withdrawal Request

**Time Required**: 15-20 minutes
**Difficulty**: Medium

### Before You Start
✓ Admin access
✓ Authority to approve withdrawals
✓ Access to payment systems
✓ Understanding of fund rules

### Step-by-Step

**Step 1: Navigate to Withdrawal Requests**
- [ ] Admin → Withdrawals
- [ ] Filter by Status: "Pending"
- [ ] See queue of requests

**Step 2: Select Request**
- [ ] Click withdrawal request
- [ ] Review details displayed

**Step 3: Verify Request Details**
- [ ] **Investor**: Name and account #
- [ ] **Fund**: Which fund
- [ ] **Amount**: Requested $
- [ ] **Type**: Partial or Full
- [ ] **Date**: When requested
- [ ] **Investor Notes**: Any special instructions

**Step 4: Eligibility Checks**
- [ ] **Lock-Up Period**:
  - [ ] Check investor start date
  - [ ] Verify lock-up terms for fund
  - [ ] Confirm period met ✅ or not met ❌
- [ ] **Available Balance**:
  - [ ] Check current position value
  - [ ] Verify available for withdrawal
  - [ ] Confirm sufficient balance ✅ or insufficient ❌
- [ ] **Bank Info**:
  - [ ] Verify bank details on file ✅ or missing ❌
  - [ ] Check account verified ✅ or not verified ❌
- [ ] **Compliance**:
  - [ ] Check for any holds ✅ clear or ❌ hold active
  - [ ] Verify KYC current ✅ or ❌ expired
  - [ ] Review recent activity for patterns

**Step 5: Make Decision**

**If All Checks Pass → APPROVE**:
1. [ ] Click "Approve" button
2. [ ] Verify approved amount = requested amount
3. [ ] Add admin notes:
   - "All eligibility checks passed"
   - "Bank information verified"
   - "No compliance concerns"
4. [ ] Click "Confirm Approval"
5. [ ] Status → "Approved"
6. [ ] Investor notified automatically

**If Partial Approval Needed**:
1. [ ] Click "Approve" button
2. [ ] Modify approved amount (less than requested)
3. [ ] Add admin notes explaining:
   - "Available balance: $X"
   - "Approved $Y per available funds"
   - "Investor may submit new request for balance"
4. [ ] Click "Confirm Approval"
5. [ ] Investor notified of partial approval

**If Rejection Needed**:
1. [ ] Click "Reject" button
2. [ ] Select rejection reason:
   - [ ] Lock-up period not met
   - [ ] Insufficient balance
   - [ ] Missing bank information
   - [ ] Compliance hold
   - [ ] Other (specify)
3. [ ] Add detailed explanation:
   - What's missing or wrong
   - How investor can resolve
   - When they can resubmit
4. [ ] Click "Confirm Rejection"
5. [ ] Investor notified with reason

**Step 6: Process Payment** (if approved)
1. [ ] Status changes to "Approved"
2. [ ] Click "Process Withdrawal"
3. [ ] Verify payment details:
   - [ ] Bank routing number
   - [ ] Account number
   - [ ] Account holder name
4. [ ] Select payment method:
   - [ ] Wire transfer
   - [ ] ACH transfer
   - [ ] Cryptocurrency
5. [ ] Click "Initiate Payment"

**Step 7: Execute Payment** (external systems)
- [ ] **For Wire/ACH**:
  - [ ] Log into bank portal
  - [ ] Initiate transfer
  - [ ] Use withdrawal ID as reference
  - [ ] Save confirmation number
- [ ] **For Crypto**:
  - [ ] Verify wallet address
  - [ ] Initiate blockchain transaction
  - [ ] Save transaction hash

**Step 8: Record Payment Details**
1. [ ] Return to withdrawal request

## 12. ADMIN: How to Update Fund NAV

**Time Required**: 15 minutes per fund
**Difficulty**: Medium

### When to Update
- **Daily**: End of business day (4:00 PM ET typical)
- **Month-End**: Critical for statement generation
- **Ad-Hoc**: After significant market moves

### Before You Start
✓ Current market prices for all holdings
✓ Updated transaction data
✓ Fee calculations
✓ Access to NAV calculation worksheet

### Step-by-Step

**Step 1: Gather Data**
- [ ] Current market prices:
  - [ ] BTC price
  - [ ] ETH price
  - [ ] USDC price (always $1.00)
  - [ ] Other assets
- [ ] Outstanding shares for fund
- [ ] Cash position
- [ ] Accrued fees
- [ ] Pending transactions

**Step 2: Calculate NAV**
- [ ] Use NAV calculation formula:

```
NAV = (Total Assets - Total Liabilities) / Shares Outstanding

Total Assets =
  + Market value of all holdings
  + Cash and equivalents
  + Accrued interest/yield
  + Other assets

Total Liabilities =
  - Accrued management fees
  - Accrued performance fees
  - Other liabilities

NAV per share = Total NAV / Shares Outstanding
```

**Example**:
```
Total Assets:
  100 BTC × $67,000 = $6,700,000
  500 ETH × $3,500 = $1,750,000
  Cash: $50,000
  Total: $8,500,000

Total Liabilities:
  Accrued fees: $12,000
  Net: $8,488,000

Shares Outstanding: 10,000
NAV per share: $8,488,000 / 10,000 = $848.80
```

**Step 3: Navigate to Fund Management**
- [ ] Admin → Funds
- [ ] Select fund to update
- [ ] Click "Update NAV"

**Step 4: Enter NAV Data**
- [ ] **Calculation Date**: [today's date]
- [ ] **NAV per Share**: [$848.80]
- [ ] **Total Assets**: [$8,500,000]
- [ ] **Total Liabilities**: [$12,000]
- [ ] **Shares Outstanding**: [10,000]
- [ ] **Data Source**: [Bloomberg, internal calculation, etc.]

**Step 5: Review Calculation**
- [ ] System shows previous NAV
- [ ] % change from previous day
- [ ] Flag if change > 5% (review carefully)
- [ ] Verify math: (Assets - Liabilities) / Shares = NAV
- [ ] Check for data entry errors

**Step 6: Add Notes** (if significant change)
- [ ] Explain large movements:
  - "BTC increased 8% today"
  - "Performance fee accrued"
  - "Large deposit processed"

**Step 7: Publish NAV**
- [ ] Review one final time
- [ ] Click "Publish NAV"
- [ ] Confirm action
- [ ] NAV updated for all investors
- [ ] Timestamp recorded

**Step 8: Verify Update**
- [ ] Check fund dashboard shows new NAV
- [ ] Verify investor dashboards updated
- [ ] Confirm NAV history recorded
- [ ] Check no error messages

**Step 9: Document**
- [ ] Save NAV calculation worksheet
- [ ] File supporting documentation:
  - [ ] Market price sources
  - [ ] Transaction confirmations
  - [ ] Fee calculations
- [ ] Update NAV log
- [ ] Archive per retention policy

### Quality Control

**Sanity Checks**:
- [ ] NAV changed by reasonable amount (<5% typical)
- [ ] Math correct (double-check calculation)
- [ ] All holdings included
- [ ] No duplicate counting
