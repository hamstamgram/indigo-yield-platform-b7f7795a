# The Self-Learning Genius Workflow (Elliot's Mandate)

This is the definitive protocol for assembling new titans into the Council. It ensures every agent is grounded in the User's specific library and issues proactive life tasks.

## Steps

1. **Knowledge Harvesting (Elliot)**:
   - **Identify Source**: Identify the core books or transcripts (e.g., from `protocol-90-curriculum.md`).
   - **Library Scan**: Elliot scans `~/Downloads` and the Apple Books library (`~/Library/Containers/com.apple.iBooksX/...`).
   - **Upload to NotebookLM**: Procured files must be uploaded to the agent's NotebookLM project for deep grounding.
   - **Index**: Log the file paths in `memory/NOTEBOOKS.md`.

2. **SOUL Generation & Grounding (Antigravity)**:
   - **Category Tagging**: Assign a functional category (`[Mindset]`, `[Physical]`, `[Wealth]`, `[Social]`, `[Systems]`).
   - **Life Rules Injection**: Every SOUL MUST include a "Life Rules" section (e.g., Rule of 21, Rule of 100) to ensure proactive task generation.
   - **Narrative Grounding**: Inject the "Experience Grounding Protocol" into the SOUL.
